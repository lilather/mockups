# Sakura Sushi - Premium Japanese Restaurant Website

A modern, responsive sushi restaurant website built with Bootstrap 5, featuring elegant animations and a sophisticated design inspired by premium Japanese dining establishments.

## 🍣 Features

- **Responsive Design**: Fully responsive layout that works perfectly on all devices
- **Modern UI/UX**: Clean, elegant design with smooth animations
- **4 Complete Pages**: Home, Menu, About, and Contact pages
- **Interactive Elements**: Tabbed menu system, contact forms, and smooth scrolling
- **Mobile Optimized**: Animations disabled on mobile for better performance
- **SEO Friendly**: Semantic HTML structure and meta tags

## 🎨 Design Inspiration

This website draws inspiration from premium sushi restaurants:

- **Sugarfish Sushi**: Clean, minimalist aesthetic
- **Kombula**: Modern typography and layout
- **Sushi Tama LA**: Sophisticated color scheme and presentation

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **Bootstrap 5**: Responsive framework
- **JavaScript (ES6+)**: Interactive functionality
- **AOS (Animate On Scroll)**: Smooth scroll animations
- **Google Fonts**: Playfair Display & Inter typography

## 🎭 Animations

- **AOS (Animate On Scroll)**: Smooth fade-in animations
- **Hover Effects**: Interactive card and button animations
- **Mobile Optimization**: Animations disabled on mobile devices (≤768px)
- **Performance**: Debounced scroll events for smooth performance

## 📱 Responsive Breakpoints

- **Mobile**: ≤768px (animations disabled)
- **Tablet**: 769px - 991px
- **Desktop**: ≥992px

## 🔧 Customization

### Colors

Modify the CSS variables in `css/style.css`:

```css
:root {
  --primary-color: #2c1810;
  --secondary-color: #8b4513;
  --accent-color: #d4af37;
  /* ... other variables */
}
```

### Animations
